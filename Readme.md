# Tracker — стильный трекер задач

Учебный проект веб-приложения для управления задачами с раздельным фронтендом и бэкендом.

## ✨ Особенности
- Полный CRUD (создание, чтение, обновление, удаление)
- Тёмный Material-дизайн с адаптивной вёрсткой
- Фильтрация задач («Все», «Активные», «Выполненные»)
- Валидация ввода и понятные сообщения об ошибках
- База данных SQLite (данные сохраняются на диск)
- Раздельная архитектура (SPA-фронтенд + REST API)

## 🛠 Стек
**Фронтенд:** HTML5, CSS3 (Flexbox, анимации), Vanilla JavaScript  
**Бэкенд:** Python 3, Flask, Flask-CORS, SQLite3

## 📁 Структура проекта
tracker/
├── backend/          # Flask-сервер и база данных
│   ├── venv/         # Виртуальное окружение (игнорируется Git)
│   ├── app.py        # Основной файл сервера
│   ├── db.py         # Работа с SQLite
│   └── tasks.db      # Файл базы данных (игнорируется Git)
├── frontend/         # Клиентская часть
│   ├── index.html    # Разметка
│   ├── style.css     # Стили (тёмная тема)
│   └── app.js        # Логика и fetch-запросы
└── README.md

## ⚙️ Быстрый старт

### 1. Бэкенд
cd backend
python3 -m venv venv
source venv/bin/activate   # Linux/Mac
# venv\Scripts\activate    # Windows
pip install flask flask-cors
python app.py
# Сервер запущен на http://127.0.0.1:5000

### 2. Фронтенд
Открой папку frontend в VS Code.
Запусти Live Server (правый клик по index.html → Open with Live Server).
Приложение откроется на http://127.0.0.1:5500.

## 📝 Примечания
- При первом запуске база данных создаётся автоматически.
- Для тестирования API используй curl (примеры в истории коммитов).
