# Трекер задач

## Бэкенд
1. Перейди в папку backend: `cd backend`
2. Активируй виртуальное окружение: `source venv/bin/activate`
3. Установи зависимости: `pip install flask flask-cors`
4. Запусти сервер: `python app.py`
Сервер будет доступен на http://127.0.0.1:5000

## Фронтенд
1. Открой папку frontend в VS Code (File → Open Folder → выбери `frontend/`)
2. Запусти Live Server (правой кнопкой на index.html → Open with Live Server)
Откроется браузер на http://127.0.0.1:5500
